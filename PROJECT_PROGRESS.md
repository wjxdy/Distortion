# 项目进度

## 基本信息
- 项目名称：《失真 Distortion》
- 当前阶段：开发早期 / MVP 垂直切片
- 最后更新：2026-06-15

## 当前状态
- 项目目标是做一个 5-8 分钟的赛博朋克像素叙事侦探 demo：开场世界观、自由审讯、探索取证、揭示第一块真相、结尾钩子。
- 核心卖点是“自由输入盘问被改写记忆的人”，用对方前后矛盾撬出真相。
- 当前仓库已经包含 Godot 4 客户端骨架和 Node.js LLM 代理后端。

## 已完成
- 设计文档已明确 MVP 范围、世界观、五天制作顺序和砍掉内容。
- Godot 客户端已有开场序章、审讯室 UI、探索取证、钥匙/关键词/真相判定和结尾文本。
- Node 后端已有 `POST /chat`，会把 Godot 对话历史转发给 OpenAI 兼容模型接口。
- 后端已有基础单元测试，覆盖 message 构建、回复解析和重试逻辑。

## 关键决策
- 2026-06-15: MVP 引擎选择 Godot 4，目标导出 HTML5 网页版。
- 2026-06-15: 审讯中的老人回复由 LLM 生成，但真相触发采用确定性规则：持有线索钥匙 + 玩家发言命中关键词。
- 2026-06-15: MVP 只做一个真相点：妻子林秀兰与蓝裙子/红烧肉相关记忆。

## 技术 / 结构备注
- 客户端位于 `Game/client`，主要脚本在 `game/` 和 `scenes/`。
- 后端位于 `Game/server`，入口是 `src/server.js`，默认监听 `8787`。
- 当前后端实现接入月之暗面 Kimi（`KIMI_API_KEY` / `KIMI_MODEL`），而设计文档早期写的是腾讯云大模型/SCF；这是需要后续确认或统一的差异。
