# 项目进度

## 基本信息
- 项目名称：《失真 Distortion》
- 当前阶段：开发早期 / MVP 垂直切片
- 最后更新：2026-06-19

## 当前状态
- 项目目标是做一个 5-8 分钟的赛博朋克像素叙事侦探 demo：开场世界观、自由审讯、探索取证、揭示第一块真相、结尾钩子。
- 核心卖点是“自由输入盘问被改写记忆的人”，用对方前后矛盾撬出真相。
- 当前仓库已经包含 Godot 4 客户端骨架和 Node.js LLM 代理后端。

## 已完成
- 设计文档已明确 MVP 范围、世界观、五天制作顺序和砍掉内容。
- Godot 客户端已有开场序章、审讯室 UI、探索取证、钥匙/关键词/真相判定和结尾文本。
- Node 后端已有 `POST /chat`，会把 Godot 对话历史转发给 OpenAI 兼容模型接口。
- 后端已有基础单元测试，覆盖 message 构建、回复解析和重试逻辑。
